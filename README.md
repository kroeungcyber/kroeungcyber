### What I Build

**AI-Assisted Service Desks** — Intelligent ticket intake using Azure OpenAI entity extraction and sentiment scoring, with adaptive routing that respects regional data residency requirements.

**Automated Incident Response** — Logic Apps and Sentinel runbooks that trigger remediation playbooks, deliver Adaptive Card notifications, and generate stakeholder briefings without human bottlenecks.

**Knowledge & Field Operations** — Governed vector search experiences with role-based access and multilingual support so field teams get answers in their language, instantly.

**Digital Workplace Governance** — Conditional Access, Purview data governance, and Defender for Cloud Apps protecting sensitive beneficiary data across distributed, high-risk environments.
