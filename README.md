## The Problem I Solve

Civil society organizations handle sensitive beneficiary data but lack the resources for enterprise security teams. I bridge that gap — designing Zero-Trust environments that scale with lean teams and meet compliance requirements NGOs actually face.

## Core Stack

Microsoft Entra ID · Microsoft 365 · Microsoft Purview · Power Automate · Azure · Splunk SIEM

---

## How I Work

**Zero-Trust Security**
Conditional Access, MFA enforcement, device compliance — without blocking the workflows NGOs depend on.

**GRC & Data Governance**
DLP policies, sensitivity labels, automated retention. Compliance mapped to real operational constraints.

**AI-Powered Operations**
Multi-agent pipelines for ticket triage, compliance monitoring, and documentation. Lean teams do more.

**Incident Response**
Threat detection, log correlation, and response playbooks. Built in Splunk with Wireshark and Nmap for training.
